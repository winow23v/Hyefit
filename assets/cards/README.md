# Card Images

Put real card images in this folder.

Use one of the following values in `card_master.card_image_url`:

- Network URL: `https://...`
- Local asset path: `assets/cards/your_card_image.png`
- Asset prefix form: `asset:assets/cards/your_card_image.png`

Recommended image ratio: around 1.58:1 (credit card ratio), PNG/JPG.
