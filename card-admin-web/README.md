# Card Admin Web

앱과 분리된 카드 마스터 관리 웹 페이지입니다.

## 1) 로컬 실행

```bash
cd card-admin-web
./start.sh
```

브라우저에서 아래 중 하나로 접속

- `http://localhost:8080`
- `http://localhost:8080/card-admin`

모바일에서 열 때는 `localhost` 대신 맥북 IP를 사용하세요.

- 예: `http://192.168.0.10:8080/card-admin`

## 2) 사용 용도

- 관리자 로그인
- CSV 업로드로 카드/티어/룰 대량 등록
- 등록된 카드 조회/삭제
- 카드명/카드사 검색 + 페이징 조회

## 3) CSV 입력 팁

- 한 카드에 혜택이 여러 개인 경우, 같은 카드/같은 티어의 다음 행에서는 카드 정보 칸을 비워도 됩니다.
- 비운 칸은 자동으로 윗행 값을 상속합니다.
- `티어순서`는 입력하지 않습니다. 최소/최대 실적 구간 기준으로 자동 정렬됩니다.
- `카테고리`가 애매하면 `계산카테고리`에 `외식/교통/쇼핑/편의점/온라인쇼핑/마트/기타` 중 하나를 직접 입력하면 됩니다.

## 4) 주의 사항

- Supabase RLS 정책에서 관리자 계정만 카드 마스터 테이블 수정 권한이 있어야 합니다.
- 운영 배포 시 `service_role` 키는 절대 브라우저 코드에 넣지 마세요.
